# Tampermonkey Userscripts

A personal collection of Tampermonkey scripts to improve web browsing. 

## Scripts Included

### [GameFAQs Guide Banner Hider](gamefaqs-banner-hider.user.js)
Permanently hides the "Would you recommend this Guide?" banner on GameFAQs pages to save screen space. 
* Uses CSS to prevent the banner from flashing on load.
* Replicates native JavaScript classes to ensure the page layout doesn't break when the banner is removed.

**Installation:** [Click here to install the script automatically](https://github.com/KlausVorutsu/gamefaqs-banner-hider/raw/refs/heads/main/gamefaqs-banner-hider.user.js) *(Requires the Tampermonkey extension)*
